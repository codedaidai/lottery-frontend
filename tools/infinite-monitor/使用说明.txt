========================================
🚀 三步启动
========================================

步骤1: 配置数据库
-----------------
编辑文件: config/database.json

修改为你的服务器信息:
{
  "host": "服务器IP地址",
  "port": 3306,
  "user": "数据库用户名",
  "password": "数据库密码",
  "database": "zspupu_shequtuan",
  "charset": "utf8mb4"
}

步骤2: 安装依赖
-----------------
打开终端，执行:
cd /Users/ywl/无限赏/infinite-monitor
pip3 install -r requirements.txt

步骤3: 启动服务
-----------------
./start.sh

或者手动启动:
cd backend
python3 app.py

========================================
📊 数据说明
========================================

库存刷新机制:
- 初始库存总和 ≈ 10000
- 每次抽奖扣减 real_pro_stock
- 当总库存 ≤ 50 时自动刷新
- 刷新公式: real_pro_stock = gailv2 × 100

赏品等级:
- 🔴 宝箱赏(5): 最稀有,概率最低
- 🟠 传说赏(39): 概率约0.1-0.5%
- 🟣 超神赏(38): 概率约0.5-1%
- 🔵 稀有赏(37): 概率约2-6%
- 🟢 普通赏(36): 概率约80-90%

========================================
⚠️ 常见问题
========================================

Q: 数据库连接失败？
A: 检查 config/database.json 配置是否正确
   确认数据库服务器允许远程连接
   检查防火墙规则

Q: 页面打不开？
A: 确认后端服务已启动
   检查端口5000是否被占用
   浏览器访问 http://localhost:5000/api/health 测试

Q: 数据不刷新？
A: 检查自动刷新开关是否开启
   点击"手动刷新"按钮测试
   查看浏览器控制台是否有错误
